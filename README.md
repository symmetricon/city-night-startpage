## Use

## Configure Links

configuration via ```JSON``` found in ```dist(or w.e you named it)/src/links-config.json```

### structure
```
{
    category": "category 1",
    linksArr: [
        {
            "linkName": "Link 1",
            "url": "https://url-for-link-1"
        },
        {
            "linkName": "Link 2",
            "url": "https://url-for-link-2"
        }
    ]
},
{
    category": "category 2",
    linksArr: [
        {
            "linkName": "Link 1",

        },
        {
            "linkName": "Link 2",
            "url": "https://url-for-link-2"
        }
    ]
}
```
